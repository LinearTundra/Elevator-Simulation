<b>Elevator Simulator</b>

A 2D elevator simulation built in Unity 6 as part of a Unity Developer Intern assignment.
Live Build
https://lineartundra534.itch.io/elevator-simulator

<br>
<b>Features</b>

3 elevators across 4 floors (Ground, 1, 2, 3)
Floor call buttons (Up/Down) per floor
Nearest available elevator dispatch logic
Elevators prioritize requests in their current travel direction
Smooth platform movement between floors
Per-elevator floor and direction display UI

<br>
<b>How to Run
From Itch.io</b>

Open the live build link above
Play in browser

<br>
<b>
  
From Source
</b>

Clone the repository
Open the project in Unity 6
Open the main scene
Press Play



<br>
<b>
Known Bugs
